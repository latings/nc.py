## nc.py

#介绍
1.实现了windows和linux的编码问题
2.真正实现了解决tcp粘包问题
3.支持多个客户端同时链接nc
4.等等...

#使用
直接下载编译后的版本

windows

# nc服务端
nc_server_windows_amd64.exe -p 8084

# nc客户端
nc_client_windows_amd64.exe -h 127.0.0.1 -p 8084

linux

# nc服务端
./nc_server_linux_amd64 -p 8084

# nc客户端
./nc_client_linux_amd64 -h 127.0.0.1 -p 8084
